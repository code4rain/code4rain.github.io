SSH 로그인시 접속지연 해결방법

09 Mar 2014

사내망의 서버나 PC와 같이 IP에 대해 DNS에 도메인 네임이 등록되지 않는 서버에 접속시

로그인 과정에서 3,4초 딜레이가 발생한다.

/etc/ssh/sshd_config 파일에 "UseDNS no"로 설정하면, 딜레이를 줄일 수 있다.

ssh 데몬 재시작이 필요. $sudo /etc/init.d/ssh restart 혹은 service ssh restart
